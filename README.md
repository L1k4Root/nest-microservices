# Como levantar el proyecto
1. Clonar el repo
2. crear un env con las variables de entorno necesarias (ver .env.template)
3. Levantar los servicios con docker-compose up --build
4. Acceder a la app en http://localhost:3010 (o el puerto que
    hayas configurado en el .env)
